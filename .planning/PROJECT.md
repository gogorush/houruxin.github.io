# Personal Blog

## What This Is

A personal blog hosted on GitHub Pages at houruxin.github.io. A clean, fast-loading website for publishing thoughts, articles, and updates with a focus on readability and simplicity.

## Core Value

Publish blog posts easily with a clean, professional reading experience.
